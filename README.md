# Reset

Eine private PWA fuer Pornofreiheit, Check-ins, Clean-Day-Counter und Notfallaktionen. Sie kann auf dem iPhone ueber Safari zum Home-Bildschirm hinzugefuegt werden.

## Lokal starten

```sh
python3 -m http.server 4173
```

Dann im Browser oeffnen:

```text
http://localhost:4173
```

Auf dem iPhone muss statt `localhost` die lokale IP-Adresse des Macs verwendet werden.
